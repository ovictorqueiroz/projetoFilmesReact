## 🚀 Tecnologias utilizadas

- **React Native** — construção da interface
- **Expo** — framework para rodar React na web e mobile
- **Firebase Realtime Database** — banco de dados em tempo real
- **Firebase Hosting** — hospedagem do app


---

## ✅ Funcionalidades

| Operação | Descrição |
|---|---|
| **Create** | Cadastro de filmes com título e diretor |
| **Read** | Listagem em tempo real de todos os filmes cadastrados |
| **Update** | Edição de título e diretor de um filme existente |
| **Delete** | Exclusão de filme com confirmação antes de deletar |
